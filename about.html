About The Project

Academic Web Development Project (v2.0)

Welcome to Taaza Khabar 2.0. This website was designed and developed as a student project to demonstrate proficiency in modern web development technologies, specifically focusing on API integration and dynamic UI updates.

"The goal of this project was to build a functional, responsive News & Weather Application that interacts with multiple real-world APIs."

This application aggregates live news from The Guardian Open Platform API and provides real-time climate data using WeatherAPI.com. It features dynamic DOM manipulation, asynchronous JavaScript (Fetch API), and a custom CSS design inspired by top-tier news publications.

Key Features Implemented:

  • Live Weather Widget (New in v2.0)
  • Real-time News Data Fetching
  • Dynamic Category Filtering
  • City-based Weather Search & Local Storage
  • Responsive Grid Layout
  • Error Handling & Loading States
